# Task 2 — Landing Page

## 📋 Description
A modern, conversion-focused landing page built with HTML, CSS, and JavaScript for a fictional AI writing tool called "NovaMind". Designed with clean sections, smooth animations, and a strong call-to-action flow.

## ✨ Features
- Hero section with animated mockup
- Logo/trust bar
- Feature cards grid
- "How It Works" 3-step process
- Pricing plans (Starter, Pro, Team)
- Customer testimonials
- Call-to-action banner
- Fully responsive footer

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ How to Run
1. Open the project folder
2. Run the following command in terminal:
npx serve . -p 3001
3. Open browser at `http://localhost:3001`
