# Task 1 — Personal Portfolio Website

## 📋 Description
A fully responsive personal portfolio website built using HTML, CSS, and JavaScript. The site showcases skills, projects, work experience, and contact information in a clean, modern design.

## ✨ Features
- Responsive navbar with mobile hamburger menu
- Hero section with animated avatar
- About Me section with stats
- Skills section with animated progress bars
- Featured Projects showcase
- Resume timeline with downloadable PDF link
- Contact form with success message
- Smooth scroll navigation

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ How to Run
1. Open the project folder
2. Run the following command in terminal:
npx serve . -p 3000
3. Open browser at `http://localhost:3000`
