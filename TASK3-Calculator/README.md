# Task 3 — Calculator

## 📋 Description
A fully functional calculator web app built with HTML, CSS, and JavaScript. Supports basic arithmetic operations with a clean dark-themed UI and calculation history.

## ✨ Features
- Addition, Subtraction, Multiplication, Division
- Percentage and sign toggle
- Calculation history panel
- Keyboard support (type numbers directly)
- Button ripple animation effect
- Responsive design for mobile and desktop

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)

## ▶️ How to Run
1. Open the project folder
2. Run the following command in terminal:
npx serve . -p 3002
3. Open browser at `http://localhost:3002`

## ⌨️ Keyboard Shortcuts
- Numbers `0-9` — Input digits
- `+ - * /` — Operators
- `Enter` — Calculate
- `Escape` — Clear all
- `Backspace` — Delete last digit
